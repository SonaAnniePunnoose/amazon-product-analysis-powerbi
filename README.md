# Amazon Product Analysis Dashboard (Power BI)

## Project Overview
This project analyzes Amazon product data to identify trends in product ratings, discounts, and customer reviews.
The dashboard provides insights into product performance across different categories.

## Tools Used
* Power BI
* Data Cleaning
* Data Visualization
* DAX

## Dataset
The dataset contains 1464 Amazon products with information including:
* Product category
* Ratings
* Number of reviews
* Discount percentage
* Actual and discounted price

## Key KPIs
* Total Products: 1464
* Average Rating: 4.10
* Average Discount: 47.71%
* Total Reviews: 27 Million

## Dashboard Insights
* Computers & Accessories category has the highest number of products.
* Most products have ratings above 4, indicating positive customer satisfaction.
* Some products receive thousands of reviews, showing high popularity.
* Higher discounts do not necessarily lead to higher product ratings.
